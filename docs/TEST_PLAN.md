# План тестирования

## Базовый функционал
1. Регистрация пользователя
   - Ввод имени, телефона/email/СНИЛС и пароля.
   - Ожидаемо: создан аккаунт, получен токен.
2. Авторизация пользователя
   - Ввод логина и пароля.
   - Ожидаемо: успешный вход, переход на главную.
3. Адаптивность
   - Проверка на ширинах 360px, 768px, 1280px.
   - Ожидаемо: элементы крупные, интерфейс не ломается.

## Проверка системы обучения
1. Запуск симуляции MAX
   - Открыть «Работа с мессенджером MAX».
   - Пройти шаг «Отправка текстового сообщения».
2. Практическое задание
   - Выполнить сценарий «Работа с мессенджером MAX» базового уровня.
   - Ожидаемо: линейное прохождение без ошибок.
3. Кнопка «Помощь»
   - Нажать «Помощь» на шаге.
   - Ожидаемо: отображается контекстная подсказка.

## Личный кабинет
1. Статистика прогресса
   - Выполнить 2–3 шага и открыть профиль.
   - Ожидаемо: корректные числовые значения и полосы прогресса.
2. Автосохранение
   - Перезагрузить страницу.
   - Ожидаемо: прогресс восстановился (локально + при наличии API).

## Дополнительные функции
1. Чат‑бот
   - Запросить подсказку через «Помощник MAX».
   - Ожидаемо: возвращается подсказка по текущему шагу.
2. Смена уровня сложности
   - Переключить базовый/расширенный.
   - Ожидаемо: шаги и прогресс адаптируются к уровню.
