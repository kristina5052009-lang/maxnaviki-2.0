# Как записать демонстрацию

## Вариант 1: Xbox Game Bar (встроено в Windows)
1. Откройте проект в браузере.
2. Нажмите `Win + G`.
3. Нажмите кнопку записи ●.
4. Запишите прохождение: вход → выбор уровня → симуляция MAX → практика → тест → кабинет.
5. Остановите запись. Видео сохранится в `Видео/Захваты`.

## Вариант 2: OBS Studio
1. Установите OBS Studio.
2. Создайте сцену и источник «Захват окна» (браузер).
3. Нажмите «Начать запись».
4. Пройдите ключевые сценарии.
5. Нажмите «Остановить запись». Видео сохранится в папке OBS.

### Советы
- Поставьте масштаб браузера 110–125% для лучшей читаемости.
- Озвучьте голосом, что вы делаете на каждом шаге.
- Запись лучше вести в 1080p, 30 fps.
