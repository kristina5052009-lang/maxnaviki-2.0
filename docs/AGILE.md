# Agile‑план (спринты)

## Спринт 1 — Базовый функционал (1–2 недели)
- Пользовательский интерфейс с крупными элементами, русификация.
- Темы: смартфон, MAX, покупки, Госуслуги.
- Линейная навигация и кнопка «Помощь».
- Выбор уровня сложности.
- Черновой REST API (курсы, прогресс).

## Спринт 2 — Симуляции и практика (1–2 недели)
- Пошаговые симуляции интерфейсов.
- Практические задания в виртуальной среде.
- Подсветка активных зон, анимация курсора.
- Озвучивание инструкций.

## Спринт 3 — Личный кабинет и качество (1–2 недели)
- Прогресс, достижения, автосохранение.
- Виртуальный помощник.
- Тесты (auth, адаптивность, симуляции).
- Подготовка демонстрации и финальной презентации.
